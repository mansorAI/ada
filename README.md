# مصروفاتي 💸

تطبيق ويب لتتبع المصروفات الشخصية — متعدد المستخدمين مع نظام مجموعات وتسجيل دخول.

🌐 **الموقع:** [masrofati.vercel.app](https://masrofati.vercel.app)

## المميزات

- **تسجيل الدخول** بالبريد الإلكتروني مع حماية كاملة عبر Supabase Auth
- **نظام المجموعات** — أنشئ مجموعتك أو انضم لمجموعة بـرمز دعوة
- **صلاحيات ثلاثية** داخل كل مجموعة: مالك / محرر / مشاهد
- **تسجيل المصروفات** يدوياً أو عبر التعرف الصوتي — منفصلة تماماً عن الالتزامات
- **فئات ديناميكية** تُنشأ تلقائياً عند إضافة أي مصروف
- **الالتزامات الشهرية** لتتبع المصاريف الثابتة مع تفاصيل الأصناف
- **جدول الأصناف** داخل كل التزام: الصنف / الفئة الملونة / الكمية / سعر الوحدة / الإجمالي / المخزون
- **تعديل الأصناف** بالضغط على أي صف في الجدول
- **متابعة المخزون** لكل صنف: كامل 🟢 / النصف 🟡 / الربع 🟠 / نفذ 🔴 / كمية مخصصة
- **تحليل الفواتير بالصورة** باستخدام Claude AI لاستخراج الأصناف تلقائياً
- **الميزانية والمتبقي** مع قائمة تفصيلية لكل المبالغ المضافة ومصادرها
- **بداية الشهر المخصصة** — حدد تاريخ بداية شهرك أو استخدم التقويم التلقائي
- **المقارنة الشهرية** للالتزامات بين الأشهر مع مؤشرات الارتفاع والانخفاض
- **أرشيف شهري** تلقائي للالتزامات المنتهية
- **تقارير وإحصائيات** مع رسوم بيانية للأسبوع والشهر وكل الوقت
- **تقرير شامل** يجمع المصروفات والالتزامات مع الميزانية والمتبقي
- **نصائح مالية** تُولَّد تلقائياً بناءً على أنماط الإنفاق
- **واجهة عربية RTL** كاملة مع خط Tajawal

## الصفحات

| الصفحة | الوصف |
|--------|-------|
| الرئيسية | إجمالي (مصروفات + التزامات منفصلَين) + ميزانية + متبقي + شريط الشهر |
| تفاصيل الفئة | كل إدخالات فئة مصروف معينة |
| الالتزامات | كروت الالتزامات + مقارنة شهرية + أرشيف |
| تفاصيل الالتزام | جدول أصناف (إجمالي + مخزون) + تقارير الفئات |
| المجموعات | إنشاء مجموعة / الانضمام برمز دعوة / التبديل بين المجموعات |
| التقارير | إحصائيات المصروفات + تقرير شامل مع الالتزامات |
| المهام | قريباً — نظام تتبع المهام اليومية |

## طريقة الاستخدام

### تسجيل الدخول
افتح [masrofati.vercel.app](https://masrofati.vercel.app) وأنشئ حساباً بالبريد الإلكتروني. عند أول دخول تُنشأ مجموعة شخصية تلقائياً.

### المجموعات
- اضغط **👤 اسمك** في الشريط العلوي
- اختر **📁 إدارة المجموعات**
- أنشئ مجموعة جديدة أو انضم برمز دعوة
- شارك **رمز الدعوة** مع من تريد لمتابعة مصروفاتك معاً

### إضافة مصروف
اضغط زر **إضافة مصروف** في شريط التنقل السفلي:
- 🎙️ **صوتي** — قل مثلاً: `"قهوة 15 ريال"` أو `"بنزين 200 ريال"`
- ✏️ **إضافة يدوي** — ادخل الاسم والمبلغ

### الالتزامات الشهرية
اذهب لصفحة **الالتزامات**، أضف فئاتك الشهرية (إيجار، كهرباء، أغراض البيت...)، ثم اضغط على أي التزام لتفتح صفحة الأصناف:

| الإجراء | الطريقة |
|---------|---------|
| إضافة صنف يدوي | اسم + فئة + كمية + سعر |
| إضافة صنف صوتي | قل: "شامبو 25 ريال" |
| تحليل فاتورة | التقط صورة → Claude يستخرج الأصناف تلقائياً |
| تعديل صنف | اضغط على الصف في الجدول |
| متابعة المخزون | اضغط على زر المخزون في الجدول |
| تسجيل الدفعة | زر "تسجيل دفعة الشهر" يحسب الإجمالي تلقائياً |

### الميزانية
- اضغط **+ إضافة مبلغ** لإضافة مبلغ للميزانية (راتب، مكافأة...)
- اضغط على **قسم الميزانية** لرؤية قائمة بكل المبالغ
- **المتبقي** = الميزانية − (المصروفات + الالتزامات)

## التقنيات المستخدمة

| التقنية | الغرض |
|---------|--------|
| HTML5 / CSS3 / JavaScript | الواجهة — لا frameworks |
| Supabase | قاعدة البيانات + تسجيل الدخول + الصلاحيات |
| Vercel | الاستضافة والنشر التلقائي |
| GitHub | إدارة الكود المصدري |
| Web Speech API | التعرف الصوتي بالعربية |
| Claude API (Haiku) | تحليل صور الفواتير |
| Google Fonts (Tajawal) | الخط العربي |

## بنية قاعدة البيانات (Supabase)

| الجدول | الوصف |
|--------|-------|
| `profiles` | ملفات المستخدمين (مرتبط بـ auth.users) |
| `groups` | المجموعات مع رمز الدعوة |
| `group_members` | أعضاء كل مجموعة مع الصلاحيات |
| `group_settings` | إعدادات الشهر والميزانية لكل مجموعة |
| `expenses` | المصروفات اليومية |
| `commitments` | الالتزامات الشهرية |
| `commitment_items` | أصناف كل التزام |
| `budget_entries` | إدخالات الميزانية |
| `audit_log` | سجل كل التعديلات |
